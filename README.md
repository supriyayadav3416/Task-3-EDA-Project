# Task 3 - Exploratory Data Analysis (EDA)

This project focuses on exploring and analyzing a dataset to discover patterns, trends, and relationships in the data.  
It is part of my data science and machine learning practice work.

---

## Project Objective

The objective of this project is to perform exploratory data analysis on the dataset and extract meaningful insights using statistical methods and visualizations.

---

## Dataset Used

- Titanic Dataset (train.csv)

---

## Tools & Libraries Used

- Python
- Pandas
- Matplotlib
- Seaborn

---

## Steps Performed

- Loaded and inspected the dataset
- Checked dataset structure and data types
- Handled missing values
- Generated statistical summaries
- Analyzed relationships between variables
- Created visualizations:
  - Correlation heatmap
  - Survival count plot
  - Gender vs Survival analysis

---

## Key Insights

- Gender has a strong influence on survival rate
- Certain features show correlation with survival outcome
- Data visualization helps in better understanding patterns

---

## Conclusion

This project helped me understand how to explore data, identify patterns, and generate insights using Python libraries.
